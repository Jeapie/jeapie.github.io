put/notifications/:id (Track open)

Definition

https://jeapie.com/api/v1/notifications/:id

Parameters
id: String Required
required Notification id
app_id: String Required
required Jeapie app id
opened: Boolean Require
required Set to "true"

Result Format

  • 200 OK
{"success": true}

Documentation

curl --include \
     --request PUT \
     --header "Content-Type: application/json" \
     --data-binary '{"opened":true}' \
     https://jeapie.com/api/v1/notifications/{notificationId}
